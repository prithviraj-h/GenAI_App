Use Case: Outlook Not Opening

Required Info:
- Operating System (Windows/Mac/Linux)
- Account Type (Office365/Exchange/IMAP)
- Error Message (if any)

Solution Steps:
- Verify internet connectivity.
  - Check Outlook version and apply latest updates.
  - Clear Outlook cache and restart system.
  - If issue persists, reconfigure Outlook profile with account details.
  - Escalate to email server admin if authentication issue continues.

Use Case: printer is not working

Required Info:
- Printer model
- Error messages (if any)
- Operating system
- Connection type (USB
- Network)
- Whether other users are experiencing the same issue

Solution Steps:
Check ink or toner levels — replace if low or empty.

Run print head or drum cleaning from printer settings.

Reinstall or update printer driver from the manufacturer’s website.

Check paper size and type settings in the printer driver.

Run the Windows Printer Troubleshooter (Settings → System → Troubleshoot).

Remove and reseat cartridges/toner properly.

If still blank, the print head may be clogged or damaged — contact service support.

Use Case: curry split on my laptop

Required Info:
- Laptop model
- Extent of the spill
- Whether the laptop is still powered on

Solution Steps:
Power off laptop immediately.

Unplug charger and remove battery (if possible).

Disconnect all accessories and peripherals.

Wipe off visible curry and moisture gently.

Turn laptop upside down (tent position) to drain.

Let it dry completely for 24–48 hours.

Clean residue carefully with a damp cloth and isopropyl alcohol.

Do not power on until fully dry.

Take laptop to a professional for internal cleaning.
